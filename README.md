# Sales Data Analysis using Python

## Project Overview
This project analyzes sales data using Python to identify trends, calculate total sales, and visualize product performance.

The analysis helps understand which products generate the highest sales and profit.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains sales information including:

- Date
- Product
- Sales
- Profit

## Project Files
sales_data_analysis
│
├── sales_data.csv
├── sales_analysis.ipynb
└── README.md

## Features
- Load sales dataset
- Calculate total sales
- Product-wise analysis
- Visualization using bar charts

## Output
The project generates insights such as:
- Total sales revenue
- Product-wise sales comparison
- Sales visualization charts
